## Installation

You can use [npm](https://nodejs.org/) or [yarn](https://yarnpkg.com/) to
install wew. wew requires Node.js 6.0.0 or above.

```sh
$ npm install --global wew
$ yarn global add wew
```

If successful, the `wew` command should show the cli documentation.
